# Ames-Housing-Prices-Regression-Analysis
The Ames, Iowa housing dataset represents the value of the sale conditions of 3000 homes with 81 variables across different categories encompassing both qualitative and quantitative variables. The objective of the assignment is to craft a regression model incorporating the techniques from the course to predict the target variable, SalePrice. The benchmark set to evaluate the model’s efficacy was to reduce the root mean squared error and assess this score through a five fold cross validation. The methodology employed was data processing and exploration to remove problematic variables and smooth the variables that are kept. Additionally, a robust variable selection to address any threats to the model’s assumption of normality by evaluating skewness and applying logarithmic transformations. The main model was selected by comparing 5 different model types and comparing Akaike Information Criterion (AIC) and RMSE and selecting the most appropriate model with a score of 0.095
